# route Module

This module handles local route graph loading, route computation, and route visualization.

## Files

- `astar.c`: Core A* route solve producing `RoutePath` outputs.
- `graph.c`: `graph.bin` loader/free logic and graph memory ownership.
- `route.c`: Route runtime state management (mode/profile, start/goal, active path set).
- `route_render.c`: Route polyline and marker drawing helpers.

## How It Connects

- Uses graph artifacts generated by offline tooling under `data/regions/<region>/graph/graph.bin`.
- Integrates with app interaction flow for node snapping, recompute, and playback.
